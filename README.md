# ADSO Project: Problem-Solving Through Responsive Web Design

## Connectivity Crises: The Digital Divide in Rural Colombia

Este proyecto es una propuesta de solución tecnológica para abordar la brecha digital y la exclusión en las zonas rurales de Colombia, desarrollada como parte del programa **Análisis y Desarrollo de Software (ADSO)** del **SENA**.

### 🚀 Objetivo del Proyecto
Analizar la falta de conectividad en la "Colombia Profunda" y proponer soluciones basadas en diseño web adaptable y tecnologías de bajo ancho de banda para mejorar el acceso a servicios esenciales como educación (SENA), salud y mercados agrícolas.

### 🛠️ Tecnologías Utilizadas
- **HTML5**: Estructura semántica y accesible.
- **CSS3**: Diseño responsivo con Glassmorphism, CSS Grid y Flexbox.
- **JavaScript**: Animaciones suaves, interactividad y simulación de formularios.
- **Mobile-First**: Optimizado para dispositivos con conexiones 2G/3G de baja velocidad.

### 📋 Entrega: Momento 1
- Identificación del problema y reporte inicial.
- Bitácora de análisis (Logbook Entry #1).
- Propuesta de tres soluciones tecnológicas inclusivas.
